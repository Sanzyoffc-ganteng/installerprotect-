# 🛡️ SANZY PROTECT - Advanced Pterodactyl Panel Protection

9-layer protection system untuk mengamankan panel Pterodactyl dari serangan internal & external.

## ✨ Features

- 🔒 **9 Layer Protection** (Configurable 0-9)
- 🛡️ **Anti-Kudeta** - Protect main admin dari dihapus/diedit
- 🔐 **Anti-Sabotase** - Lock critical settings (nodes, settings, etc)
- 👥 **Anti-User Tampering** - Admin tidak bisa edit user lain
- 🚫 **Anti-Server Delete** - Block penghapusan server
- 🔑 **Anti-2FA Manipulation** - Proteksi 2FA
- 📁 **Anti-File Access** - Block akses file server orang lain
- 👁️ **Anti-Server Snooping** - Block intip server lain
- 🎯 **Whitelist Admin** - Kontrol pembuatan admin baru
- 🔍 **Auto-Filter API** - Filter response API per user
- 📊 **Beautiful Dashboard** - UI management di admin panel
- 📝 **Logging** - Audit trail semua blocked attempts
